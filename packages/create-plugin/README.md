# Create WP Plugin

CLI tool to scaffold WordPress plugin with simple block

## Installation

Install the CLI tool globally

```bash
npm install @wordpress/create-plugin -g
```

Using tool to scaffold plugin/block

```bash
create-wp-plugin
```
